Game Rules

1. EACH ARMY RECEIVES AN EQUAL NUMBER OF CARDS

2. DURING EACH BATTLE EACH ARMY FLIPS A CARD FROM THEIR DECK.

3. THE ARMY WITH THE HIGHEST CARD VALUE WINS THE BATTLE.

4. IF TWO ARMIES HAVE THE SAME CARD VALUE, THEY GO TO WAR.
WAR... WHAT IS IT GOOD FOR.
5. IF ALL TROOPS ARE SEIZED FROM AN ARMY...
THEY LOSE THE WAR AND ARE... REMOVED

6. ONCE AN ARMY HAS SEIZED ALL TROOPS, THEY WIN
GOOD LUCK GENERAL...
